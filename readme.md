Answer to the questions


1. getElementById() is to get any element by calling it's id name.
   getElementByClass() is to get elements which have the same class name.
   querySelector() is to get the first element of any list or matching things by calling it's id, class name,tag etc.
   querySelectorAll() is to get the nodeList of matching things by calling it's id, class name,tag etc.


2. I use createElement() and appendChild() accordingly to create and insert a new element into the DOM.

3. When any event starts from the target element to it's parent elements is called event bubbling.
